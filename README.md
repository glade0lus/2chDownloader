# 2chDownloader  
## About  
Command-line tool for downloading images and videos from 2ch.hk  

## How to compile  
* Install CMake and Boost  
* Clone repository  
* Clone submodules: `git submodule update --init --recursive`  
* Compile: `cmake . && make`
